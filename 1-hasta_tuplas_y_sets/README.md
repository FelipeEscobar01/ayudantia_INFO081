## 🍉 **Ejercicio 1**

```

```

La función **retornará si estos tres datos forman o no** un trio pitagorico.

## 🍉 **Ejercicio 4**

```

```

Esta función **recibe la tupla** y **retorna la moda** presente en el conjunto de datos.
